== Description ==
The plugin enables the store orders to be automatically completed depending on various conditions. The plugin is compatible with the latest version of WooCommerce & works with all the major payment gateway including WooCommerce payment, PayPal, Stripe and many more.

=== ✨ Features of Autocomplete Orders for WooCommerce ===
* All orders of any products: Orders status will be changed as “Completed” upon successful payment.
* All orders of any products: Orders status will be changed as “Completed” depends on BANK TRANSFER, CHECK, CASH ON DELIVERY & Others payment processor.
* Paid orders of virtual products only: orders containing “Virtual” products only will be completed upon successful payment.
* Paid orders of downloadable products only: orders containing “Downloadable” products only will be completed upon successful payment.

=== WooCommerce Autocomplete Orders ===
The WooCommerce Autocomplete Orders plugin offers a wide range of features that provide complete control over the order completion process.

* Autocomplete orders for Direct Bank Transfer payments
* Autocomplete orders for Check payments
* Autocomplete orders for Cash on Delivery payments
* Autocomplete orders for Cash on Delivery orders with downloadable products

== Installation ==
Install this plugin and automatically complete your WooCommerce orders. This plugin is compatible with the latest version of WooCommerce & works with all the major payment gateway including WooCommerce payment, PayPal, Stripe and many more.
1. Upload the plugin files to the `/wp-content/plugins/autocomplete-orders-for-woocommerce` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Navigate to WooCommerce > Settings > General > General options > Autocomplete Orders section in your Dashboard
4. Chose the preferable option from the dropdown list how you want your orders to be autocompleted.
